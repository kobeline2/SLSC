本稿では, SLSC に基づく適合度評価を6分布に拡張したモンテカルロ計算と, X 空間 / S 空間の区別に基づく理論整理を通じて再検討した. 
得られた結論は以下の通りである.

\begin{enumerate}
  \item SLSC の標本数依存性は, 既往研究で示された傾向がより広い分布群に対しても維持される. 
正しい分布を当てはめた場合, SLSC は標本数の増加とともに概ね $N^{-1/2}$ のスケールで小さくなる.
  \item SLSC が変換に対して不変となるのは線形変換の場合に限られる. 
よって, 非線形標準化された SLSC は, 元の SLSC と同値な指標ではない.
  \item LN3を対数標準化した S 空間の SLSC は, 真の分布が他の分布である場合にも小さい値を与えやすく, モデル比較を系統的にゆがめる傾向を持つ.
\end{enumerate}

以上より, SLSC を用いる場合には, 固定閾値による一律判定を避けるとともに, 比較は X 空間で統一的に行う必要がある. 
非線形標準化を伴う SLSC を, 他の分布と同列の適合度指標として用いることは適当でない.

今後の課題としては, LN3そのものの統計的性質の整理, および SLSC に代わる尤度や情報量規準に基づく分布選択法を進めることが挙げられる.
